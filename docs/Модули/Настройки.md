# Настройки

Фронтенд: `SettingsPage.jsx` (`/admin/settings`).

API: `/api/settings` — `backend/src/routes/settings.js`.

Доступ: чтение — все авторизованные; запись — только `admin`.

## Хранение

Таблица `settings_s` — key-value хранилище:

| Поле | Тип | Описание |
|---|---|---|
| key | VARCHAR(100) | Уникальный ключ |
| value | TEXT | Значение (строка) |

API:
- `GET /api/settings` — все настройки как объект `{key: value}`
- `PUT /api/settings` — UPSERT нескольких ключей (тело: `{key1: value1, key2: value2}`)

## Разделы настроек

Страница настроек разделена на вкладки:

### Сканирование
| Ключ | Описание | По умолчанию |
|---|---|---|
| `scan_fast_threshold` | Порог быстрого скана (сек) | - |
| `scan_slow_threshold` | Порог медленного скана (сек) | - |
| `scan_sound_enabled` | Звук включён | - |
| `scan_sound_freq_ok` | Частота звука успеха (Гц) | - |
| `scan_sound_freq_err` | Частота звука ошибки (Гц) | - |
| `scan_sound_dur_ok` | Длительность звука успеха (мс) | - |
| `scan_sound_dur_err` | Длительность звука ошибки (мс) | - |
| `scan_auto_delay` | Задержка авто-скана (мс) | - |
| `scan_min_length` | Мин. длина штрихкода (символов) | - |

### Интерфейс
| Ключ | Описание |
|---|---|
| `default_box_size` | Размер коробки по умолчанию (шт) |
| `products_page_size` | Количество товаров на странице |
| `products_row_density` | Плотность строк таблицы |
| `toast_duration_success` | Длительность уведомления успеха (сек) |
| `toast_duration_error` | Длительность уведомления ошибки (сек) |

### Аналитика инвентаризации
| Ключ | Описание |
|---|---|
| `inventory_fresh_hours` | Порог свежести (часов) |
| `inventory_stale_hours` | Порог просрочки (часов) |
| `inventory_color_fresh` | Цвет свежего (hex) |
| `inventory_color_warn` | Цвет предупреждения (hex) |
| `inventory_color_stale` | Цвет просроченного (hex) |

### GRACoin
| Ключ | Описание |
|---|---|
| `gra_inventory_scan_rate` | Ставка за сканирование (GRA/единица, default: 10) |

### Другие вкладки
- **Внешний вид** — тема, цвета
- **Данные** — проверка на маркетплейсах (WB, OZON)
- **История** — changelog версий
- **О системе** — информация о системе

## Контекст настроек на фронтенде

Настройки загружаются через `AppSettingsContext` и доступны во всех компонентах через `useAppSettings()`.
