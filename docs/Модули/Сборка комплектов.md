# Сборка комплектов (Assembly)

Фронтенд (admin): создание через `TasksPage.jsx` (`/admin/tasks`).
Фронтенд (employee): `AssemblyPage.jsx` (`/employee/assembly/:id`).

API: `/api/assembly` — `backend/src/routes/assembly.js`.

## Концепция

Сборка комплектов — задача типа `bundle_assembly`. Сотрудник собирает готовые наборы (бандлы) из отдельных товаров-компонентов.

Пример: комплект "Магний L-треонат 2 банки" = 2 баночки Магния.

## Фазы (assembly_phase)

```mermaid
graph LR
    picking[Забор товара] --> assembling[Сборка комплектов]
    assembling --> placing[Размещение]
    placing --> done[Завершено]
```

### 1. Picking (забор)
- Сотрудник забирает компоненты с паллетов/полок
- Сканирует место (паллет/полку), затем коробку (если паллет), затем баночки
- Количество в источнике уменьшается

### 2. Assembling (сборка)
- Сканирует компоненты в каждый комплект
- Печатает штрих-код комплекта
- Подтверждает сканом наклеенного ШК

### 3. Placing (размещение)
- Сканирует место назначения (полка/паллет)
- Сканирует каждый собранный комплект
- Количество в назначении увеличивается

## Таблицы

### inventory_tasks_s (дополнительные поля)
| Поле | Описание |
|---|---|
| task_type | `bundle_assembly` |
| bundle_product_id | FK products_s — товар-бандл |
| bundle_qty | Сколько комплектов собрать |
| assembly_phase | picking / assembling / placing / done |
| source_boxes | JSONB — откуда брать компоненты |
| dest_shelf_id | FK shelves_s — куда положить |
| dest_pallet_id | FK pallets_s — куда положить |
| assembled_count | Сколько комплектов собрано |
| placed_count | Сколько комплектов размещено |

### assembly_items_s
| Поле | Описание |
|---|---|
| task_id | FK inventory_tasks_s |
| product_id | FK products_s — компонент |
| source_box_id | FK boxes_s — откуда взяли |
| source_pallet_id | FK pallets_s — откуда взяли |
| source_shelf_id | FK shelves_s — откуда взяли |
| scanned_barcode | Отсканированный ШК |
| quantity | Количество |
| used_in_bundle | Номер комплекта (0 = не привязан) |

## API эндпоинты

| Метод | Путь | Описание |
|---|---|---|
| GET | /source-locations | Где хранятся компоненты бандла |
| POST | / | Создать задачу сборки |
| GET | /:id | Детали задачи |
| GET | /:id/source-boxes | Коробки-источники |
| POST | /:id/start-picking | Начать забор |
| POST | /:id/scan-pick | Сканировать товар при заборе |
| POST | /:id/start-assembling | Начать сборку |
| POST | /:id/scan-component | Сканировать компонент |
| POST | /:id/confirm-bundle | Подтвердить комплект (скан ШК) |
| POST | /:id/start-placing | Начать размещение |
| POST | /:id/scan-place | Сканировать комплект при размещении |
| POST | /:id/complete | Завершить задачу |
| DELETE | /:id | Удалить задачу (откат перемещений) |
