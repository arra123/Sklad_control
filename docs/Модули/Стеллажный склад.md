# Стеллажный склад

## Иерархия

```
Склад (warehouse) → Стеллаж (rack) → Полка (shelf)
                                         ├── Россыпь (shelf_items)
                                         └── Коробки (shelf_boxes)
                                              └── Товары (shelf_box_items)
```

## Типы складов

| warehouse_type | Описание |
|---|---|
| `stelajsklad` | Стеллажный склад |
| `visual` | Визуальный стеллажный склад |
| `both` | Комбинированный |

## Штрихкоды

- У стеллажей: `racks_s.barcode_value`
- У полок: `shelves_s.barcode_value`
- У коробок на полках: `shelf_boxes_s.barcode_value`

Сотрудник сканирует штрихкод полки для начала инвентаризации или перемещения.

## Визуальный склад

Графическое представление структуры: стеллажи и полки в виде сетки. Компонент `FBSVisualView.jsx`.

## Аудит

Все операции логируются в `shelf_movements_s`:
- `inventory` — результат инвентаризации
- `correction` — ручная корректировка
- `stock_in` / `stock_out` — приход/расход
