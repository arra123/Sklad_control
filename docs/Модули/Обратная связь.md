# Обратная связь (Feedback)

Фронтенд: `FeedbackButton.jsx` (кнопка на каждой странице), `FeedbackModal.jsx` (форма отправки).

API: `/api/feedback` — `backend/src/routes/feedback.js`.

## Концепция

Сотрудники могут отправить обратную связь прямо из приложения — баг-репорт, предложение, вопрос. Поддерживается загрузка скриншотов и аудио.

## Таблица feedback_s

| Поле | Тип | Описание |
|---|---|---|
| user_id | INTEGER | Кто отправил |
| username | VARCHAR | Логин отправителя |
| user_role | VARCHAR | Роль отправителя |
| category | VARCHAR | Категория (bug/suggestion/question) |
| subcategory | VARCHAR | Подкатегория |
| description | TEXT | Описание |
| transcript | TEXT | Транскрипция аудио |
| screenshot_path | VARCHAR | Путь к скриншоту |
| audio_path | VARCHAR | Путь к аудио |
| page_url | VARCHAR | Страница, с которой отправлено |
| browser_info | TEXT | Информация о браузере |
| status | VARCHAR | new / reviewed / resolved |
| admin_notes | TEXT | Заметки админа |
| resolved_by | INTEGER | Кто обработал |
| resolved_at | TIMESTAMPTZ | Когда обработано |

## API эндпоинты

| Метод | Путь | Доступ | Описание |
|---|---|---|---|
| POST | / | auth | Создать фидбэк (multipart/form-data) |
| GET | / | admin | Список фидбэка |
| GET | /:id | admin | Детали фидбэка |
| PATCH | /:id | admin | Обновить статус/заметки |
| DELETE | /:id | admin | Удалить фидбэк |
