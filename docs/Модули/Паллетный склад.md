# Паллетный склад

## Иерархия

```
Склад (warehouse) → Ряд (pallet_row) → Паллет (pallet)
                                            ├── Россыпь (pallet_items)
                                            └── Коробки (boxes)
                                                 └── Товары (box_items)
```

## Типы складов

| warehouse_type | Описание |
|---|---|
| `paletsklad` | Паллетный склад |
| `visual_pallet` | Визуальный паллетный склад |

## Коробки (boxes_s)

| Статус | Описание |
|---|---|
| `open` | Коробка открыта, можно добавлять товары |
| `closed` | Коробка закрыта и подтверждена |

- `box_size` — максимальное кол-во товаров в коробке
- `is_remainder` — неполная коробка (остаток)
- `confirmed` — подтверждена после упаковки

## Визуальный склад

Компонент `FBOVisualView.jsx` — графическое отображение рядов и паллетов.
