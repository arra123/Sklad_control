# Claude Code Skills (Slash Commands)

Установлены **21 команда** из 3 популярных репозиториев. Файлы лежат в `.claude/commands/`.

## Дебаг и фиксы

| Команда | Что делает |
|---------|-----------|
| `/smart-debug` | Умный дебаг с трассировкой и контекстом |
| `/smart-fix` | Найти и исправить баг автоматически |
| `/debug-trace` | Трассировка ошибок через стек вызовов |
| `/error-analysis` | Глубокий анализ ошибок |

## Код-ревью и рефакторинг

| Команда | Что делает |
|---------|-----------|
| `/full-review` | Полный код-ревью проекта |
| `/refactor-clean` | Рефакторинг с проверкой |
| `/code-explain` | Объяснение кода (для онбординга) |
| `/feature-development` | Разработка фичи по шагам |

## Деплой

| Команда | Что делает |

## Как использовать

В чате Claude Code набрать `/` и название команды:
```
/smart-debug
/deploy-checklist
/security-scan
```

Файлы команд: `.claude/commands/*.md` — можно редактировать и добавлять свои.
