# Claude Code Skills (Slash Commands)

Файлы лежат в `.claude/commands/`.

## Дебаг и фиксы

| Команда | Что делает |
|---------|-----------|
| `/smart-debug` | Умный дебаг с трассировкой и контекстом |
| `/smart-fix` | Найти и исправить баг автоматически |
| `/error-analysis` | Глубокий анализ ошибок |

## Код-ревью и рефакторинг

| Команда | Что делает |
|---------|-----------|
| `/full-review` | Полный код-ревью проекта |
| `/refactor-clean` | Рефакторинг с проверкой |
| `/feature-development` | Разработка фичи по шагам |

## Деплой

| Команда | Что делает |
|---------|-----------|
| `/deploy-checklist` | Чеклист перед деплоем |
| `/deploy-prepare` | Подготовка релиза (версия, changelog) |
| `/deploy-hotfix` | Хотфикс деплой |
| `/deploy-rollback` | Откат деплоя |

## Безопасность

| Команда | Что делает |
|---------|-----------|
| `/security-audit` | Полный аудит безопасности |

## Производительность

| Команда | Что делает |
|---------|-----------|
| `/perf-audit` | Аудит производительности |
| `/performance-optimization` | Оптимизация (бандл, запросы, рендер) |

## Контекст

| Команда | Что делает |
|---------|-----------|
| `/dream` | Консолидация памяти: сканирует сессию и сохраняет в persistent memory |

## Как использовать

В чате Claude Code набрать `/` и название команды:
```
/smart-debug
/deploy-checklist
/security-audit
```

Файлы команд: `.claude/commands/*.md` — можно редактировать и добавлять свои.
