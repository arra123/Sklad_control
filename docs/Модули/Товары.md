# Товары

## Каталог

Товары хранятся в `products_s`, импортируются из [[МойСклад API]] или создаются вручную.

Фронтенд: `ProductsPage.jsx` (`/admin/products/cards`), `ProductStockPage.jsx` (`/admin/products/stock`).

## Типы товаров

| entity_type | Описание |
|---|---|
| `product` | Обычный товар |
| `bundle` | Бандл (комплект из нескольких товаров) |

## Ключевые поля products_s

| Поле | Тип | Описание |
|---|---|---|
| `external_id` | VARCHAR(255) | ID из МойСклад |
| `name` | VARCHAR(500) | Название |
| `code` | VARCHAR(255) | Код товара |
| `article` | VARCHAR(255) | Артикул |
| `barcode_list` | TEXT | Штрихкоды через `;` |
| `production_barcode` | VARCHAR(255) | Производственный штрихкод (начинается с 46) |
| `marketplace_barcodes_json` | JSONB | Штрихкоды маркетплейсов `[{type, value}]` |
| `stock` | NUMERIC | Остаток |
| `reserve` | NUMERIC | Резерв |
| `in_transit` | NUMERIC | В пути |
| `quantity` | NUMERIC | Общее количество (stock + in_transit) |
| `folder_id` | INTEGER | FK → product_folders_s |
| `archived` | BOOLEAN | Архивный |
| `sale_price` | NUMERIC(18,2) | Цена продажи |
| `cost_price` | NUMERIC(18,2) | Себестоимость |
| `source_json` | JSONB | Полные данные из МойСклад |

## Штрихкоды

У товара может быть несколько штрихкодов:
- `barcode_list` — все штрихкоды (TEXT, через `;`)
- `production_barcode` — производственный штрихкод (формат `46XXXXXXXXXXX`)
- `marketplace_barcodes_json` — штрихкоды маркетплейсов (JSONB массив)

Маркетплейсы: Wildberries (`20*`, `40*`), Ozon (`OZN*`, `Z*`), Yandex Market, СберМегаМаркет.

Управление: `POST /:id/barcode`, `DELETE /:id/barcode`, `PUT /:id/barcode-type`.

## Бандлы

Бандл — товар-комплект. Состав хранится в `bundle_components_s`:
- `bundle_id` → товар-бандл (FK products_s)
- `component_id` → товар-компонент (FK products_s)
- `component_external_id` — ID компонента в МойСклад
- `quantity` — количество компонента

При просмотре бандла компоненты загружаются из `bundle_components_s`, с fallback на `source_json` (МойСклад формат).

## Папки

Иерархия папок из МойСклад: `product_folders_s` с `parent_id` для вложенности и `full_path` для отображения.

## Проверка на маркетплейсах

- [[Wildberries API]] — `POST /api/products/wb-check` (поиск по штрихкоду)
- OZON — `POST /api/products/check-ozon` (проверка по магазину), `POST /api/products/check-ozon-all` (все товары)
- OZON магазины: `ozon_1` (ИП И.), `ozon_2` (ИП Е.)

## Синхронизация

`POST /api/products/sync` — запуск импорта каталога из [[МойСклад API]].
История: `GET /api/products/import/history` → `import_runs_s`.
