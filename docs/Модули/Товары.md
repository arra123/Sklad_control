# Товары

## Каталог

Товары хранятся в `products_s`, импортируются из [[МойСклад API]].

## Типы товаров

| entity_type | Описание |
|---|---|
| `product` | Обычный товар |
| `bundle` | Бандл (комплект из нескольких товаров) |

## Штрихкоды

У товара может быть несколько штрихкодов:
- `barcode_list` — основные штрихкоды (TEXT, через запятую)
- `production_barcode` — производственный штрихкод
- `marketplace_barcodes_json` — штрихкоды маркетплейсов (JSONB)

Маркетплейсы: Wildberries, Ozon, Yandex Market, СберМегаМаркет.

## Бандлы

Бандл — товар-комплект. Состав хранится в `bundle_components_s`:
- `bundle_id` → товар-бандл
- `component_id` → товар-компонент
- `quantity` → количество компонента

## Папки

Иерархия папок из МойСклад: `product_folders_s` с `parent_id` для вложенности.
