# Перемещения (Movements)

Фронтенд (admin): `MovementsPage.jsx` (`/admin/movements`).
Фронтенд (employee): `MovePage.jsx` (`/employee/move`), `MyInventoryPage.jsx` (`/employee/inventory`).

API: `/api/movements` — `backend/src/routes/movements.js`.

## Универсальный лог

Таблица `movements_s` — аудит всех перемещений товаров в системе.

Ключевые поля: `movement_type`, `product_id`, `quantity`, `quantity_before`, `quantity_after`, `from_*` / `to_*` (pallet_id, shelf_id, box_id, employee_id), `performed_by`, `source`, `notes`.

## Типы перемещений

### Между локациями
| Тип | Описание |
|---|---|
| shelf_to_shelf | Полка → Полка |
| shelf_to_pallet | Полка → Паллет |
| pallet_to_shelf | Паллет → Полка |
| pallet_to_pallet | Паллет → Паллет |
| box_to_box | Коробка → Коробка |

### С участием сотрудника
| Тип | Описание |
|---|---|
| shelf_to_employee | Полка → Сотрудник (взял) |
| employee_to_shelf | Сотрудник → Полка (вернул) |
| pallet_to_employee | Паллет → Сотрудник |
| employee_to_pallet | Сотрудник → Паллет |
| box_to_employee | Коробка → Сотрудник |
| employee_to_box | Сотрудник → Коробка |

### Системные
| Тип | Описание |
|---|---|
| external_receipt | Внешний приход |
| external_to_pallet | Добавление товара на паллет |
| correction | Ручная корректировка |
| pallet_correction_in | Увеличение кол-ва на паллете |
| pallet_correction_out | Уменьшение кол-ва на паллете |
| write_off | Списание |
| manual_correction | Корректировка инвентаря сотрудника |

### Источники (source)
`scan`, `manual_edit`, `task`, `admin`, `packaging`.

## Workflow сканирования перемещений

1. `POST /api/movements/scan` — распознать штрихкод (определяет тип: паллет, полка, коробка, товар)
2. `POST /api/movements/move` — выполнить перемещение (from → to)

## Инвентарь сотрудников

Сотрудник может "держать" товары (`employee_inventory_s`).
Workflow: взял с полки → перенёс → положил на паллет.

| Эндпоинт | Описание |
|---|---|
| `GET /my-inventory` | Мой инвентарь (текущий пользователь) |
| `GET /employee-inventory/:employeeId` | Инвентарь конкретного сотрудника |
| `GET /all-employee-inventory` | Инвентарь всех сотрудников (admin) |
| `POST /employee-inventory/:employeeId` | Добавить товар в инвентарь (admin) |
| `PUT /employee-inventory/:employeeId/:productId` | Обновить количество (admin) |
| `DELETE /employee-inventory/:employeeId/:productId` | Удалить из инвентаря (admin) |
