# Словарь терминов

## Склад и хранение

| Термин                           | Описание                                                    |
| -------------------------------- | ----------------------------------------------------------- |
| **Стеллажный склад**             | Склад со стеллажами и полками                               |
| **Паллетный склад**              | Склад с рядами и паллетами                                  |
| **Комбинированный склад (Both)** | Склад с обоими типами: стеллажи + паллеты                   |
| **Стеллаж (Rack)**               | Вертикальная конструкция с полками                          |
| **Полка (Shelf)**                | Горизонтальная поверхность на стеллаже                      |
| **Паллет (Pallet)**              | Поддон для хранения коробок/товаров                         |
| **Ряд (Row)**                    | Группа паллетов в паллетном складе                          |
| **Коробка (Box)**                | Физическая коробка на паллете или полке                     |
| **Визуальный склад**             | Графическое представление структуры склада                  |
| **Остаток (Remainder)**          | Неполная коробка после упаковки                             |

## Товары и каталог

| Термин | Описание |
|--------|----------|
| **Единичный товар (Product)** | Одна позиция каталога, например «NMN 60 капсул» |
| **Бандл / Комплект (Bundle)** | Товар-комплект из нескольких единичных товаров |
| **Артикул (Article)** | Уникальный код товара внутри компании |
| **SKU** | Stock Keeping Unit — единица складского учёта |
| **Штрихкод (Barcode)** | Уникальный код для идентификации товара/полки/паллета |
| **Производственный ШК** | Штрихкод начинающийся на 46 (российский) |
| **Маркетплейсный ШК** | Штрихкод маркетплейса (Ozon — OZN, WB — 20/40) |

## Сырьё и производство

| Термин | Описание |
|--------|----------|
| **Сырьё (Raw material)** | Исходный компонент для производства — порошки, вещества |
| **Порошок** | Группа сырья: активные вещества в граммах (куркумин, МКЦ, диоксид кремния) |
| **Готовая смесь (Mix)** | Промежуточный продукт: смесь порошков в определённых пропорциях, измеряется в граммах |
| **Полуфабрикат (Semi-product)** | Готовая капсула = пустая капсула + готовая смесь, измеряется в штуках |
| **Пустая капсула** | Желатиновая или растительная оболочка для наполнения смесью |
| **Расходники (Supplies)** | Банки, крышки, мембраны, флаконы — тара для готового продукта |
| **Этикетка (Label)** | Промышленная наклейка на банку с информацией о товаре |
| **Мембрана** | Защитная плёнка под крышку банки |

## Тех. карты и рецепты

| Термин | Описание |
|--------|----------|
| **Тех. карта (Tech card)** | Рецепт производства товара: список материалов и их количество |
| **Рецепт материала** | Из чего состоит материал (полуфабрикат → смесь + капсулы, смесь → порошки) |
| **Замес** | Одна партия смеси, например 22.7 г готовой смеси магний цитрат |
| **Выход (Output quantity)** | Сколько единиц получается из одного замеса/партии |

### Цепочка производства

```
Порошки (г) → Готовая смесь (г) → Полуфабрикат (шт) → Товар (банка)
```

**Пример: Магний цитрат**

| Этап | Материалы | Количество |
|------|-----------|-----------|
| Смесь (22.7 г) | Сырьё магния цитрат | 15.7 г |
| | Сырьё Диоксид Кремния | 7 г |
| Полуфабрикат (60 шт) | Готовая смесь | 22.7 г |
| | Пустые капсулы | 60 шт |
| Товар (1 банка) | Полуфабрикат | 60 шт |
| | Банка ПЭТ | 1 шт |
| | Крышка | 1 шт |
| | Мембрана | 1 шт |
| | Этикетка | 1 шт |

## Задачи и процессы

| Термин | Описание |
|--------|----------|
| **Инвентаризация** | Пересчёт товаров на полке/паллете через сканирование |
| **Упаковка (Packaging)** | Процесс создания коробок, сканирования товаров, размещения на паллет |
| **Сборка комплектов (Assembly)** | Процесс сборки бандлов из компонентов: забор → сборка → размещение |
| **Перемещение (Movement)** | Перенос товара между локациями (полка, паллет, коробка, сотрудник) |
| **Пик (Pick)** | Одно сканирование/взятие товара при сборке заказа |
| **Фаза забора (Picking)** | Этап assembly: забор компонентов с паллетов/полок |
| **Фаза сборки (Assembling)** | Этап assembly: сканирование компонентов в комплекты, печать ШК |
| **Фаза размещения (Placing)** | Этап assembly: размещение собранных комплектов на полку/паллет |

## Заработок

| Термин | Описание |
|--------|----------|
| **GRACoin (GRA)** | Внутренняя валюта, начисляется сотрудникам за работу |
| **Ставка (Rate)** | Сколько GRA начисляется за 1 единицу работы (скан/пик) |
| **inventory_scan** | Заработок за сканирование при инвентаризации (склад) |
| **external_order_pick** | Заработок за пик при сборке заказа (сборка-сайт) |
| **manual_adjustment** | Ручная корректировка баланса администратором |

## Интеграции

| Термин | Описание |
|--------|----------|
| **МойСклад (МС)** | Внешняя ERP-система, источник данных по товарам и сырью |
| **WB (Wildberries)** | Маркетплейс, проверка штрихкодов через Content API |
| **Ozon** | Маркетплейс, проверка штрихкодов через API |
| **Сборка-сайт** | Внешний сайт для сборки заказов OZ/WB, пишет заработок в общую БД |
| **external_id** | UUID товара/материала из МойСклад для привязки между системами |

## Технические термины

| Термин | Описание |
|--------|----------|
| **PM2** | Менеджер процессов Node.js на сервере |
| **JWT** | JSON Web Token — токен авторизации |
| **Суффикс `_s`** | Все таблицы в БД заканчиваются на `_s` (sklad) |
| **Деплой** | Автоматический через GitHub Actions → SSH → PM2 |
