# GRAсклад — Warehouse Management System

**Компания:** GRAflab (производство БАДов)
**URL:** http://147.45.97.155/sklad
**Порт:** 3017
**Версия:** v2.46.1

## Стек

| Слой | Технологии |
|---|---|
| Frontend | React 18 + Vite + Tailwind CSS + Lucide Icons |
| Backend | Node.js + Express + pg + JWT + bcryptjs |
| БД | PostgreSQL 5.42.100.180:5432, database "bd2" |
| Деплой | GitHub Actions → SSH → PM2 (process "c-site") |

## Роли

| Роль | Доступ |
|---|---|
| **admin** | Полный доступ ко всем разделам |
| **manager** | Дашборд, товары (просмотр), склады, задачи, сотрудники (просмотр), перемещения, аналитика, ошибки |
| **employee** | Мои задачи, сканирование, перемещения, мой инвентарь |
