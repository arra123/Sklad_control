# MCP серверы

> Model Context Protocol — серверы, расширяющие возможности Claude Code.

## Установленные серверы (13 шт.)

### 1. windows-mcp
**Что делает:** Полный контроль над Windows десктопом — клики, скриншоты, ввод текста, скролл, управление окнами, процессами, реестром, уведомления.

**Когда использовать:**
- Нужно проверить как выглядит приложение на экране — сделать скриншот
- Автоматизировать рутину: открыть программу, нажать кнопки, заполнить формы
- Работа с приложениями, у которых нет CLI/API (только GUI)

---

### 2. github
**Что делает:** Работа с GitHub через API — Pull Requests, Issues, репозитории, ветки, коммиты, код-ревью.

**Когда использовать:**
- Создать/посмотреть/прокомментировать Pull Request
- Управление Issues — создать баг, закрыть задачу
- Посмотреть историю коммитов, сравнить ветки

---

### 3. filesystem
**Что делает:** Прямой доступ к файловой системе — чтение, запись, создание, удаление, перемещение файлов и папок.

**Когда использовать:**
- Массовые операции с файлами
- Просмотр структуры директорий
- Получение метаданных файлов

---

### 4. git
**Что делает:** Git операции — add, commit, branch, checkout, diff, log, status, show, reset.

**Когда использовать:**
- Работа с ветками и коммитами
- Просмотр диффов и истории
- Создание веток и коммитов

---

### 5. memory
**Что делает:** Граф знаний — хранит сущности (люди, проекты, концепции) и связи между ними. Данные переживают перезапуск сессии.

**Когда использовать:**
- Запомнить контекст проекта между сессиями
- Хранить информацию о людях, ролях, решениях

---

### 6. puppeteer
**Что делает:** Браузерная автоматизация через headless Chrome. Навигация, клики, ввод текста, скриншоты, выполнение JavaScript.

**Когда использовать:**
- Быстрая проверка UI — скриншот страницы
- Тестирование формы
- Парсинг данных с веб-страниц

---

### 7. context7
**Что делает:** Загружает актуальную документацию библиотек прямо в контекст.

**Когда использовать:**
- Работаешь с API React/Express/Tailwind — нужна свежая дока
- Проверить актуальные параметры функции/хука
- Синтаксис изменился с последнего обновления

---

### 8. fetch
**Что делает:** HTTP-запросы к любым URL. GET, POST и другие методы.

**Когда использовать:**
- Проверить что API сервера отвечает
- Загрузить JSON из внешнего сервиса
- Тестирование API эндпоинтов

---

### 9. playwright
**Что делает:** Продвинутая браузерная автоматизация от Microsoft. Chrome, Firefox, Safari. Навигация, клики, скриншоты, drag-and-drop, перехват сетевых запросов.

**Когда использовать:**
- E2E тестирование
- Сложные сценарии — drag-and-drop, загрузка файлов
- Перехват и анализ сетевых запросов

---

### 10. google-drive
**Что делает:** Доступ к Google Drive — поиск файлов.

**Когда использовать:**
- Найти файл на Drive
- Загрузить данные из Google Sheets

---

### 11. sequential-thinking
**Что делает:** Пошаговый анализ сложных задач. Разбивает проблему на шаги, последовательно рассуждает.

**Когда использовать:**
- Архитектурные решения
- Сложная отладка
- Планирование фичи

---

### 12. notion
**Что делает:** Работа с Notion — страницы, блоки, базы данных, комментарии, поиск.

**Когда использовать:**
- Создание/редактирование страниц в Notion
- Запросы к базам данных Notion
- Поиск по Notion workspace

---

### 13. ide
**Что делает:** Интеграция с IDE — выполнение кода и получение диагностики.

**Когда использовать:**
- Получить список ошибок/предупреждений в файле
- Выполнить фрагмент кода

---

## Матрица: задача → сервер

| Задача | Лучший MCP | Альтернатива |
|---|---|---|
| Проверка UI после деплоя | **playwright** | puppeteer |
| Скриншот экрана Windows | **windows-mcp** | — |
| Работа с PR/Issues | **github** | — |
| Тест API эндпоинта | **fetch** | puppeteer |
| Актуальная документация | **context7** | fetch |
| Автоматизация браузера | **playwright** | puppeteer |
| Автоматизация десктопа | **windows-mcp** | — |
| Работа с Google Drive | **google-drive** | — |
| Планирование архитектуры | **sequential-thinking** | — |
| Запомнить контекст | **memory** | — |
| Массовая работа с файлами | **filesystem** | — |
| Работа с Notion | **notion** | — |
| Git операции | **git** | — |

## Управление серверами

```bash
# Список и статус
claude mcp list

# Добавить сервер
claude mcp add <имя> -- <команда>

# Удалить сервер
claude mcp remove <имя>
```
