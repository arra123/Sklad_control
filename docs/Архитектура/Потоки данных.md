## Процесс инвентаризации

```mermaid
sequenceDiagram
    participant A as 👤 Админ
    participant S as 👷 Сотрудник
    participant APP as 📱 Приложение
    participant DB as 🗄️ База данных
    participant COIN as 💰 GRACoin

    A->>APP: Создать задачу инвентаризации
    APP->>DB: INSERT inventory_tasks_s
    Note over DB: Статус: new

    S->>APP: Начать задачу
    APP->>DB: UPDATE status → in_progress

    loop Сканирование товаров
        S->>APP: Сканирует ШК товара
        APP->>DB: INSERT task_scans_s
        APP->>DB: Проверка: товар существует?
        alt Товар найден
            APP->>COIN: Начислить GRA
            COIN->>DB: INSERT employee_earnings_s
            APP-->>S: ✅ Успешно (+0.5 GRA)
        else Не найден
            APP->>DB: INSERT scan_errors_s
            APP-->>S: ❌ Ошибка
        end
    end

    S->>APP: Завершить задачу
    APP->>DB: UPDATE status → completed
    APP-->>A: 📊 Отчёт готов
```
