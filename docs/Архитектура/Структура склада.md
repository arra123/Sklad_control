## Статусы коробок

```mermaid
stateDiagram-v2
    [*] --> open: Создана
    open --> closed: Закрыта (оприходована)
    closed --> open: Вскрыта
    closed --> moved: Перемещена на другой паллет

    state open {
        [*] --> scanning: Сканирование товаров
        scanning --> scanning: + товар
    }

    state closed {
        [*] --> on_pallet: На паллете
        on_pallet --> in_transit: В пути
    }
```

## Процесс перемещения

```mermaid
graph LR
    subgraph FROM["📤 Откуда"]
        F_SHELF["Полка С3П1"]
        F_PALLET["Паллет 2"]
        F_BOX["Коробка #419"]
    end

    subgraph ACTION["⚡ Действие"]
        SCAN_FROM["Скан источника"]
        SCAN_ITEM["Скан товара"]
        SCAN_TO["Скан назначения"]
    end

    subgraph TO["📥 Куда"]
        T_SHELF["Полка С9П2"]
        T_PALLET["Паллет 5"]
        T_BOX["Новая коробка"]
    end

    subgraph LOG["📝 Лог"]
        MOVE["movements_s<br/>тип, кол-во, время,<br/>сотрудник"]
    end

    F_SHELF --> SCAN_FROM
    F_PALLET --> SCAN_FROM
    F_BOX --> SCAN_FROM
    SCAN_FROM --> SCAN_ITEM
    SCAN_ITEM --> SCAN_TO
    SCAN_TO --> T_SHELF
    SCAN_TO --> T_PALLET
    SCAN_TO --> T_BOX
    SCAN_TO --> MOVE

    style FROM fill:#fee2e2,stroke:#dc2626
    style ACTION fill:#fef3c7,stroke:#d97706
    style TO fill:#d1fae5,stroke:#059669
    style LOG fill:#dbeafe,stroke:#2563eb
```
